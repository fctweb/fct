氹仔坊眾學校
Escola Fong Chong da Taipa
01 // ORIGIN // 1966

街坊互助精神

1966年，為解決離島子弟就學困難，氹仔居民自發籌款創立本校。憑藉愛國愛澳精神，點燃了氹仔坊眾學校的全人教育星火，半個世紀以來始終是社區文化的堅實地標。

02 // KINDERGARTEN

K1-K3 啟蒙探究

打破教室邊界，將社區化作最生動的課堂。帶領幼兒走進街市認識蔬果、前往公園親近自然。透過趣味編程啟蒙與榨果汁等生活實作，在快樂探究中建立認知與獨立人格。

03 // PRIMARY

P1-P6 多元底座

全面推行「閱讀閱開心」計劃，夯實語文基礎。實施英語與數學分組分層教學，確保每位學子皆能跟上進度。透過多元社團活動與《基本法》推廣，培養五育並重的社會小公民。

04 // SECONDARY

走班學習與智慧平台

初中部率先實施「走班學習」模式，精準對接學生能力。全面引入智慧教學平台，大數據實時監測學習軌跡，提供個性化輔導方案，賦能高效自主學習。

05 // STEAM & AI

人工智能與創客體系

紮根未來，從小學到中學系統性引入 AI 與 STEAM 課程。在專屬 Maker Space 創客實驗室中，學子將理論化為實物，於國際發明展與全國機器人大賽中屢奪金獎。

06 // PHILOSOPHY

明禮進取·全人發展

秉持「明禮、進取、勤奮、創新」校訓。德育為先，透過豐富的社會實踐培養責任感；英文拔尖班輔導雅思達標，為學生搭建通往國際頂尖學府的橋樑。

07 // SYSTEM LOGS

全校榮耀矩陣

探索 FCT 歷年來在科技、學術與體育領域的輝煌成就。
